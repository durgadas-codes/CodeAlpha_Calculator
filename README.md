# CodeAlpha_Calculator
# 🧮 Calculator Web App

A simple and responsive **Calculator** built using **HTML, CSS, and JavaScript**. This project performs basic arithmetic operations with a clean user interface and real-time results.

---

## 🚀 Features

* ➕ Addition
* ➖ Subtraction
* ✖️ Multiplication
* ➗ Division
* 🧹 Clear screen (AC button)
* ⌨️ Keyboard support (Bonus feature)
* ⚡ Real-time input and result display
* 🎨 Attractive and responsive UI

---

## 🛠️ Technologies Used

* **HTML** – Structure of the calculator
* **CSS** – Styling and layout
* **JavaScript** – Functionality and logic

---

calculator-project/

## 💻 How to Run the Project

1. Download or clone the repository

   ```
   git clone https://github.com/your-username/calculator-project.git
   ```

2. Open the project folder

3. Run the `index.html` file in your browser

---

## 🎯 How It Works

* User clicks buttons or uses keyboard input
* Input is displayed on the screen
* JavaScript processes the expression
* Result is calculated and shown instantly

---

## ⌨️ Keyboard Support

| Key       | Action            |
| --------- | ----------------- |
| 0–9       | Numbers           |
| + - * /   | Operations        |
| Enter     | Calculate result  |
| Backspace | Delete last input |
| Escape    | Clear screen      |
